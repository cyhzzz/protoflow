## 填写指南

### 页面ID命名规则
- 使用小写字母、数字、连字符（-）
- 首字符必须是字母
- 示例：home, user-profile, product-detail-123

### 组件类型选择
- 框架组件：statusBar, navBar, safeArea, tabBar
- 内容组件：grid, list, card, carousel
- 表单组件：input, picker
- 反馈组件：modal, toast, actionSheet
- 通用组件：text, image, button

### Action类型选择
- 页面跳转：navigateTo, switchTab, back
- 弹窗控制：showModal, hideModal
- 反馈显示：showToast, showActionSheet
- 网络请求：request
- 状态更新：updateState

### Grid行列配置
- rowCount: 行数（如2表示2行）
- colCount: 列数（如4表示4列）
- 8个宫格：rowCount=2, colCount=4

### List cell类型
- text: 纯文本行
- iconText: 图标+文本（带箭头/开关）
- avatarText: 头像+文本（如用户列表）
